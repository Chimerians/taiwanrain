# 台灣雨 TaiwanRain 🌧️

Taiwan's cutest hyperlocal rain forecast app — powered by CWA Open Data.

## Features
- Animated radar map of Taiwan
- Rain countdown: "17分鐘後下雨"
- 8-hour hourly forecast
- Push notification alerts
- Typhoon warnings overlay
- Kawaii mascots (raindrop, typhoon frog, sunny bear)
- Dark mode
- LINE / IG share cards
- Free + Premium (NT$30/year)

## Data Sources
- CWA Open Data API: https://opendata.cwa.gov.tw
- Radar: O-A0059-001
- Forecasts: F-D0047-* (all 22 counties)
- Typhoon: W-C0034-001

## Deploy
Static site — deploy directly on Vercel, Netlify, or any CDN.

## Tech
HTML + CSS + Vanilla JS + Leaflet.js (no build step required)
